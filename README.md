# Retool AI Agent Chat Component

A custom Retool component that provides enhanced AI chat functionality with interactive widgets and rich content rendering. This project explores building seamless integrations between AI agents and Retool's low-code platform, enabling more dynamic and intelligent internal tools.

## Key Features
*   Interactive chat interface with support for AI agent responses
*   Rich content rendering for markdown, code blocks, and structured data
*   Seamless integration with Retool apps and backend queries
*   Customizable UI to match application themes

## Tech Stack
*   React
*   TypeScript
*   Retool Component SDK
*   CSS Modules

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/retool-ai-agent-chat.git`
2.  Install dependencies: `npm install`
3.  Run the development server: `npm start`
